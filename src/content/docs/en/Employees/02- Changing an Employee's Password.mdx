### Changing an Employee's Password

To change an Employee's password, follow these steps:

1. Go to the **Employees** Tab.

2. Select the Employee you want to change the password for and click on the **Lock** icon.

![](/employee/emp-8.webp)

3. You will be redirected to the Change Password window. Choose a password that matches the validation requirements and confirm it on the second text box.

<Aside type="note">
    Passwords must contain 8 characters or more, a lowercase letter, an uppercase letter, a number, and a special chatacter.
</Aside>

![](/employee/emp-9.webp)

4. Click on the **Save** button to save the new password.

<Aside type="note">
    When changing a password, the system will log you out and you will need to log in again using the new password.
</Aside>
