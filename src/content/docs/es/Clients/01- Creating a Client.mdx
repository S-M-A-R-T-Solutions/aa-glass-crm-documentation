Los clientes son las personas o empresas con las que haces negocios. Para esta entidad, el sistema solo permite crearlos como personas, no como empresas. Esto te permite tener una persona física y no una organización como punto de contacto entre AA Glass, el contratista y el cliente.

### Crear un Cliente

Para crear un cliente, sigue estos pasos:

1. En la pestaña de Clientes, haz clic en el botón "+ Agregar un Cliente".

![](/client/clients-2.webp)

2. Aparecerá un modal. Completa los campos con la información del cliente.

![](/client/clients-3.webp)

3. Selecciona el contratista que estará asociado con el cliente haciendo clic en el campo de Contratista.

![](/client/clients-4.webp)

<Aside type="note">
    Cada cliente necesita estar asociado con un contratista. Esta versión no incluye la creación de contratistas desde este selector modal, por lo que debes crear el contratista primero. Para más información, consulta la <a href="/es/contractors/01--creating-a-contractor/">sección de Creación de Contratistas</a>.
</Aside>

4. Agrega la fuente de cliente (Lead Source) haciendo clic en el menú desplegable de Fuente de Cliente.

![](/client/clients-5.webp)

5. Completa el resto de los campos con la información del cliente.

![](/client/clients-6.webp)

<Aside type="tip">
    Cuanta más información de contacto sobre el cliente puedas recopilar, mejor. Esto te ayudará a tener una mejor comunicación con el cliente.
</Aside>

6. Haz clic en el botón "Guardar" para guardar el cliente.


### Detalles del Cliente

Después de crear un cliente, puedes ver los detalles del cliente haciendo clic en el nombre del cliente en la lista de Clientes.

![](/client/clients-8.webp)
